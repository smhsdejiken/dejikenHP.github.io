#　ブログ作成について

1.文章を準備します
2.文章をBLOGcleate.htmlにペーストします
3.完成したものを新しいブログとしてフォルダに入れてください
4.新しいブログのタイトルとURLをBLOG/BL-js/blogs.jsonに追加してください

## BLOGcleateについて

css:BLOG-cleate.css
js :BLOG-cleate.js

基本操作は書いてある通りです
タブを閉じれば入力したものは消えます。文章は事前に準備してください
必要であれば自分で足りない色等を付け足してください

###　作成したファイルについて

作成時
css:BLOG-contents.css
js :BLOG-home.js

記事の管理をしやすくするために、blog（数字）.htmlの形でファイル名を付けてください

###　BLOGcodeについて

css:BLOG-cleate.css
js :BLOG-code.js

作成したものはBLOGcleate.htmlで作成したものに後から付け足してください

## BLOG-homeについて

css:BLOG-home.css
js1:BLOG-home.js
js2:BLOG-jsonTest.js

blogs.jsonの内容を下から表示します
ヘッダー部分を流用することをおススメします
一番下のデジ研の文字を押してファイル名を入力するとそのファイルに飛びます

### BLOG-searchについて

css:BLOG-contents.css
js :BLOG-search.js

テキストボックスに入力した言葉に当てはまる記事を表示します

## 部活紹介について

css:About.css
js :About.js

cssで表示、非表示を変えて、画面の中身が見えないように出来ます。
